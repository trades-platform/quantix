# 均线突破策略（ma_breakout）

## 策略逻辑

价格突破均线买入，跌破均线卖出。使用 90% 资金建仓，100 股整数倍。

```
入场：price > MA(period)
出场：price < MA(period)
```

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period | 20 | 均线周期 |

```bash
quantix backtest run-file strategies/ma_breakout.py 588000.SH 1970-01-01 2026-04-18 \
  --period 1D --adjust hfq --params '{"period":60}'
```

## 分析

### 优势

- 逻辑简单，信号明确
- MA(20) 在趋势行情中捕捉能力较强
- 相比双均线交叉，信号更及时，不会错过短趋势

### 缺陷

1. **胜率极低**（23%）：70%+ 的交易是亏损的，完全依赖少数大趋势回本
2. **震荡市反复止损**：价格在均线上下反复穿越，每次触发买卖，手续费持续消耗
3. **无止损机制**：跌破均线才出场，持仓在下跌过程中持续缩水
4. **满仓进出**：不根据信号强度或波动率调整仓位
5. **参数不稳定**：MA(20) 在本标的上最优，换个标的未必如此

### 改进方向

- 加 ADX 趋势强度过滤（ADX > 25 才开仓）
- 加 ATR 止损/追踪止损限制单笔亏损
- 突破时要求成交量高于均量确认
- 仓位按波动率倒数调整（高波动自动减仓）
